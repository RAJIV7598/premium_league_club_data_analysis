# ⚽ Premier League Club Data Analysis

## 📌 Project Overview
This project analyzes Premier League clubs based on performance metrics like:
- Win %
- Loss %
- Goals scored
- Clean sheets
- number of times club went for finals and won the title

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Key Insights
- identified elite clubs that are best to invest 
- Identified most efficient clubs based on win ratio
- Compared teams using scatter plots and bubble charts
- Highlighted high-performing teams with fewer matches
